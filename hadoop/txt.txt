Sure 😄 — here’s a **clean, step-by-step procedure** for installing Hadoop on Ubuntu using a pre-built script and running a WordCount MapReduce job.

---

## **Procedure: Install Hadoop and Run WordCount**

---

### **Step 1: Install Java**

Hadoop requires Java. Install OpenJDK 8:

```bash
sudo apt update
sudo apt install openjdk-8-jdk -y
java -version
```

---

### **Step 2: Install Hadoop using Pre-Built Script**

This automates downloading, configuring, and starting Hadoop.

```bash
sudo apt update && sudo apt install wget -y
wget https://raw.githubusercontent.com/aryanur1/Hadoop-Ubuntu-Installer/main/hadoop-installer.sh
chmod +x hadoop-installer.sh
./hadoop-installer.sh
```

✅ What this script does:

* Installs Java (if missing)
* Downloads Hadoop
* Configures environment variables
* Sets up single-node SSH
* Formats HDFS and starts Hadoop services

---

### **Step 3: Start Hadoop Services**

After installation, run:

```bash
start-dfs.sh
start-yarn.sh
jps
```

You should see processes like:

```
NameNode
DataNode
ResourceManager
NodeManager
SecondaryNameNode
```

---

### **Step 4: Create WordCount Script**

1. Create a script file:

```bash
nano run-wordcount.sh
```

2. Paste the following content:

```bash
#!/bin/bash

# WordCount Demo Script for Hadoop

echo "🔹 Checking Hadoop Services..."
jps

echo "🔹 Creating HDFS input directory..."
hdfs dfs -mkdir -p /user/hadoop/input

echo "🔹 Adding sample text file to HDFS..."
echo "Hello Hadoop Hello World. Hadoop is awesome. Big Data with Hadoop!" > input.txt
hdfs dfs -put -f input.txt /user/hadoop/input/

echo "🔹 Removing old output directory (if exists)..."
hdfs dfs -rm -r -f /user/hadoop/output

echo "🔹 Running WordCount MapReduce example..."
hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-*.jar wordcount /user/hadoop/input /user/hadoop/output

echo "🔹 Job completed. Output files:"
hdfs dfs -ls /user/hadoop/output

echo "🔹 Displaying WordCount result:"
hdfs dfs -cat /user/hadoop/output/part-r-00000

echo "✅ WordCount job finished successfully!"
```

3. Save and exit (`Ctrl + O`, `Enter`, `Ctrl + X`).

---

### **Step 5: Make Script Executable**

```bash
chmod +x run-wordcount.sh
```

---

### **Step 6: Run WordCount Job**

```bash
./run-wordcount.sh
```

---

### **Step 7: Expected Output**

* Hadoop services running (`jps` output)
* HDFS input directory created
* WordCount MapReduce job completes successfully
* Output in HDFS `/user/hadoop/output`
* Word counts displayed in terminal, e.g.:

```
Big 1
Data 1
Hadoop 3
Hello 2
World. 1
awesome. 1
is 1
with 1
```

---

### ✅ **Notes**

* This uses **single-node Hadoop** setup (pseudo-distributed mode).
* You can replace `input.txt` with any text file for other MapReduce experiments.
* No need to manually edit XML configuration files — the installer handles it.
